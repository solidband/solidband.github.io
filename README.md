[solid.band](http://solid.band)
===============================

Sources of Solid Band official website.

Building
========

    make

or:

    npm install
    coffee src/build.coffee

Debugging
=========

    make serve

or:

    coffee src/serve.coffee

and open URL in browser: http://localhost:3100/
